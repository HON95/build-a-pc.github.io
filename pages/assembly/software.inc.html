Introduction

Most operating systems, including Windows and the most common Linux distrubutions have very easy different installation instructions, which changes once in a while. Therefore I will not write too specific instructions on this, but instead briefly guide you.

Installing Windows

Windows costs a bit but most of the programs you might want to use supports it (like Microsoft Office and most games) (Linux often has a lot of open alternatives, though) and might be considered easier to use for some. If you're going for Windows 10, purchase it somewhere (you might find some cheap shops online) and download the media download program from here. This program downloads Windows and helps you burn it to a disc drive or a USB flash drive. Then just boot your new PC from this drive.

Installing Linux

The best bast of Linux is that it's open and most distros are free (as in the price is $0). If you're new to Linux, I recomment Linux Mint or Ubuntu. Both have very simple installation instructions, which include downloading some disk image and burning it to a DVD or USB flash drive.
