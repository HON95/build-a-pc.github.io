Windows

Windows is a operating system developed by Microsoft. During the growing popularity of graphical user interfaces(GUI) in 1985, Microsoft and Windows found its place in the market. Since then, Windows have continually been updated and new versions have been released every few years. Today, Microsoft Windows has grown to own more than 90% of the market share.

Windows 1.0 was the very first version and it was released in 1985, one year after Mac OS. Windows 1.0 featured components like calculator, calender, clock, control panel, notepad, paint and some other features. In Windows 1.0, only the modal dialog boxes could overlap windows. When Windows 2.0 came in 1987 there were many improvements, among these were keyboard shortcuts, user interface overhaul and overlapping windows.

Here is a list over all the different versions of Windows:

  1. Windows 1.0 (1985)
  2. Windows 2.0 (1987)
  3. Windows 3.0, 3.1 and 3.2 (1990, 1992, 1994)
  4. Windows NT 3.1, NT 3.5, NT 4 (1993, 1994, 1996)
  5. Windows 95 (1995)
  6. Windows 98 (1998)
  7. Windows 2000 (2000)
  8. Windows XP (2001)
  9. Windows XP Professional x64 (2005)
  10. Windows Vista (2007)
  11. Windows 7 (2009)
  12. Windows 8, 8.1 (2012, 2013)
  13. Windows 10 (2015)

Microsoft has not only produced the Windows operating system for computers, but they have also produced mobile versions. Although the mobile versions are not the most popular on the market, it is the third most popular, behind Android and iOS
